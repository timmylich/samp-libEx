# Что это?

**libEx** - библиотека, чьей целью является собрать внутри себя различный функционал любого направления, который может понадобится при разработке игрового мода.  
Например функции и наборы функций, лежащие на форумах, либо которые каждый разработчик создаёт самостоятельно.
